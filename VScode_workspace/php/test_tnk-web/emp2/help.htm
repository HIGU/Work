データの表示

select       from       ;

     　表示したいフィールドを指定します。すべてを表示したい場合は"*"を入力してください。

     　表示したいフィールドがあるテーブルを指定します。

例1：　select * from user_detailes;

　　　↑user_detailesテーブルからすべてのフィールドを表示します。

例2：　select uid,name from user_detailes;

　　　↑user_detailesテーブルからuidとnameというフィールドを表示します。

　

フィールドの入力内容を変更する

update       set      =      where      ;

       変更したいフィールドがあるテーブルを指定します。

     　変更したいフィールドを指定します。

     　変更する内容を入力。空白にしたい場合には"NULL"を入力してください。

     　行（変更するユーザー）を指定します。これを入力しない場合、すべての行に変更した内容が適用されるので注意。


例1：　update user_capacity set cid=2 where uid=1;

　　　↑user_capacityテーブルでuidが1のユーザーのcidを2に変更します。

例2：　update user_detailes set info=NULL where uid='000000';

　　　↑user_detailesテーブルでuidが1のユーザーのinfoを空白に変更します。

　

データを登録する

insert into       values (     ,'     ') ;

       データを登録するテーブルを指定します。

     　登録する内容を入力。文字列は' 'で囲む必要があります。

例： insert into section_master values (8,'生産部',1);

　　　↑section_master テーブルのsidフィールドに8、section_nameフィールドに"生産部"、sflgフィールドに1というデータを登録します。

　

行を削除する

delete from       where      ;

       削除したいフィールドがあるテーブルを指定します。

     　行を指定します。これを入力しない場合、すべての行に変更した内容が適用されるので注意。

例： delete from user_detailes where uid='000000';

　　　↑user_detailes テーブルのuidが1の行を削除します。

　

パターンマッチング検索

select       from       where     like       ;

       表示したいフィールドがあるテーブルを指定します。

    　フィールドを指定します。

     　文字列を入力。文字列は'% %'で囲む必要があります。

例1：select * from user_detailes where address like '%宇都宮%';

　　↑user_detailes テーブルのaddressフィールドの中に'宇都宮'という文字列がある行すべてを表示します。

例2：select * from user_detailes where name like '%木%' and uid like '%0%';

　　↑user_detailes テーブルのnameフィールドの中に'木'という文字列があり、なおかつuidの中に'0'という文字列が含まれる行すべてを表示します。

　

※：　ユーザーを削除する場合はデータベース操作からではなく、ユーザーの情報修正にある[情報の抹消]から行ってください。

※：　１度に複数のフィールドを指定する場合、フィールドとフィールドの間に","を入れてください。

※：　１度に複数のクエリーを実行する場合、クエリーとクエリーの間に";"を入れてください。
