最近ＴＶでみたんですけど、年金の記録を付けるために派遣社員を起用したらしいですね。

その起用した派遣社員が中国のお方らしいのですが、年金の記録の際にミスを多発したらしいですよ。

中国のお方の言い分では、

「ワタシ、漢字難しいアルヨー。」

みたいなこと言ってたとか、なんとか。

と、まぁ、漢字の　「見」　を　「貝」で書いたり、　「日東　太郎（仮）」を　「日　東太郎」にしてたみたいですね。

そのミスが発覚して、最終的にはやり直したらしいですが。

年金問題が取り上げられてるのに、このミスは痛いですなぁ。

漢字って以外に難しいカンジなんですかね？（うまいこと言った！）
